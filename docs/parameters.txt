1. initTransaction
--------------------------------------------------------------
In parameters:
Name						Description
address						the address of the initiate
privateKey					the private key of the initiate
recipient					the address of the recipient
amount						the amount of NEM (xem)
message						the message included into the transaction
host						specify the server's host
port						specify the server's port
Example:
java -jar initTransaction.jar -address xxx -privateKey xxx -recipient xxx -amount 15 -message "I am tester002" -host 127.0.0.1 -port 7890

Out parameters (JSON):
flag						success or error
transactionHash				the hash of transaction
Example:
{"flag":"success","transactionHash":"xxx"}



2. initMultisigTransaction
--------------------------------------------------------------
In parameters:
Name						Description
multisigAddress				the address of multisig account
cosignatoryAddress			the address of cosignatory account
cosignatoryPrivateKey		the private key of cosignatory account
recipient					the address of the recipient
amount						the amount of NEM (xem)
message						the message included into the transaction
host						specify the server's host
port						specify the server's port
Example:
java -jar initMultisigTransaction.jar -multisigAddress xxx -cosignatoryAddress xxx -cosignatoryPrivateKey xxx -recipient xxx -amount 100 -message "I am test003" -host 127.0.0.1 -port 7890

Out parameters (JSON):
flag						success or error
transactionHash				the hash of transaction
innerTransactionHash		the hash of inner transaction
Example:
{"flag":"success","transactionHash":"xxx","innerTransactionHash":"xxx"}

4. monitorIncomingTransaction
--------------------------------------------------------------
In parameters:
Name						Description
address						the address of account
host						specify the server's host
port						specify the server's port
Example:
java -jar monitorIncomingTransaction.jar -address xxx -host 127.0.0.1 -port 7890

Out parameters (JSON):
sender						the address of sender
date						the date when the transaction was created
amount						the amount of NEM (xem)
message						the message included into the transaction
isMultisig					0: is not multisig transaction, 0: is multisig transaction
Example:
{"sender":"xxx","amount":111,"date":"2017-03-14 16:57:00","message":"I am test003","isMultisig":"1"}



5. monitorIncomingTransaction
--------------------------------------------------------------
In parameters:
Name						Description
address						the address of account
host						specify the server's host
port						specify the server's port
Example:
java -jar monitorMultisigTransaction.jar -address xxx -host 127.0.0.1 -port 7890
